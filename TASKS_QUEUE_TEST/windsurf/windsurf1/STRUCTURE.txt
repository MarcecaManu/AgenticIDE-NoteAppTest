Task Flow:
----------

1. SUBMIT → Task created with PENDING status
2. EXECUTE → Status changes to RUNNING
3. PROGRESS → Progress updates (0-100%)
4. COMPLETE → Status changes to SUCCESS/FAILED
5. OPTIONAL → Cancel (PENDING only) or Retry (FAILED only)


Task Statuses:
--------------

PENDING    → Task queued, waiting to execute
RUNNING    → Task currently executing
SUCCESS    → Task completed successfully
FAILED     → Task failed with error
CANCELLED  → Task was cancelled

Quick Commands:
---------------

Start Backend:
  cd backend
  uvicorn main:app --reload --host 0.0.0.0 --port 8000

Start Frontend:
  Open frontend/index.html in browser
  OR
  cd frontend
  python -m http.server 8080

Run Tests:
  cd tests
  pytest test_tasks.py -v

Install Dependencies:
  cd backend
  pip install -r requirements.txt


API Endpoints:
--------------

POST   /api/tasks/submit          Submit new task
GET    /api/tasks/                List all tasks (with filters)
GET    /api/tasks/{task_id}       Get specific task
DELETE /api/tasks/{task_id}       Cancel pending task
POST   /api/tasks/{task_id}/retry Retry failed task
